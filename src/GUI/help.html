Polynomial / Parametric Graph Plotter

Welcome to the Graph Plotter Help.

This application allows you to plot polynomial and parametric functions.

Polynomial Functions

In the Polynomial Function tab, you can insert your desired polynomial function and calculate the plot by pressing the Calculate button. To Enter a Polynomial Function, the following syntax has to be used: Exponentials: x^2...x^3; Coefficients: 2x....2/5x...-3x

You can create up to three functions simultaneously. To add a function, click the Add button at the bottom of the window.

For better visibility, you can choose a color for each function. To delete a function, you can either remove its input area by pressing the Delete button or simply recalculate the plot by adding a new function and pressing Calculate.

To get more Information about a Graph an Information window on the bottom will appear that holds The Roots, the Extreme Points and the current Graph-color.

It is also possible to derive a function by clicking its corresponding derive button.

Parametric Functions

In the Parametric Function tab, you can add one parametric function along with its x and y components. You need to set the range in which the function will be displayed.

You can reset your inputs by clicking the Reset button, which will clear all the input fields. Alternatively, you can modify the inputs and recalculate the plot.

Options

On the upper right is a button with a gear icon. This opens an option menu where things like the grid or specific calculations can be turned off.

Screenshot and Help Button

In both tabs, you will find a Screenshot button that allows you to capture the current plot. The Help button at the bottom of the window displays this Help window.
