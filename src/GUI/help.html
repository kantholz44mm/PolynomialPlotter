Guidance for Polynomial and Parametric Graph Plotter

Welcome to the instruction manual for our Graph Plotter tool.

This innovative tool enables users to generate and visualize polynomial and parametric function graphs.

Utilizing Polynomial Functions

In the Polynomial Function section, input your chosen polynomial function, then calculate its graph using the 'Calculate' button. Note the following syntax conventions:

  • The variable must be 'x'.
  • For exponentials use the '^' symbol, for example, "x^2...x^3".
  • For coefficients, you can use any real number, a fraction, or a mathematical expression (without variable 'x'), for example, "2x...2/5x...-3x...sin(4)x...(-1/2+1/3)x^2".
  • Use '+' or '-' to add or subtract terms. Parentheses can be used to group coefficients, but not to group terms.
  • A standalone constant term can be entered without 'x', for example, "+3...-2".
  • The terms do not need to be in any particular order.
  • An empty term or a term with just 'x' will be treated as having a coefficient of 1, for example, "x...x^2...+x...-x".

You can graphically represent up to three functions concurrently. To add another function, simply click the 'Add' button located beneath the interface.

For enhanced visual clarity, each function can be assigned a unique color. To remove a function, either press the 'Delete' button to erase its input field, or add a new function and click 'Calculate' to refresh the plot.

An informative panel will appear beneath the graph, providing details such as Roots, Extreme Points, and the currently applied Graph color.

The application also enables the derivation of a function via its corresponding 'Derive' button.

If the polynomial you entered is not valid, an error message will be displayed to help you correct it. Please check the syntax and try again.

Utilizing Parametric Functions

In the Parametric Function section, you can add one parametric function with its respective x and y components. Make sure to set the range in which the function will be rendered.

The 'Reset' button can be utilized to clear all inputs. Alternatively, you can edit the inputs and click 'Calculate' to recompute the plot.

Configuration Options

In the top-right corner of the interface, you'll find a button adorned with a gear icon. Clicking this button opens the options menu, where you can adjust preferences such as grid display or specific calculations.

Screenshot and Assistance Features

Both function tabs include a 'Screenshot' button to capture the current plot, and a 'Help' button located at the bottom of the window that opens this Assistance window.
